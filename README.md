# gconv

### VCF to PED/MAP

    gconv --vcf in.vcf --out out.ped

### VCF to HapMap

    gconv --vcf in.vcf --out out.hmp

### PED/MAP to VCF

    gconv --ped in.ped --out out.vcf

### PED/MAP to HapMap

    gconv --ped in.ped --out out.hmp

### HapMap to VCF

    gconv --hmp in.hmp --out out.vcf

### HapMap to PED/MAP

    gconv --hmp in.hmp --out out.ped
